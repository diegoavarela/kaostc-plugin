# Contributing to KaosTC Plugin

Gracias por tu interés en contribuir a KaosTC Plugin.

## Cómo contribuir

### 1. Fork del repositorio

```bash
# En GitHub, click en "Fork"
# Luego cloná tu fork
git clone https://github.com/TU_USUARIO/kaostc-plugin.git
cd kaostc-plugin
```

### 2. Crear una rama para tu cambio

```bash
git checkout -b feature/mi-nueva-feature
# o
git checkout -b fix/bug-que-arregle
```

### 3. Hacer tus cambios

- Seguí la estructura existente del proyecto
- Comandos van en `commands/`
- Agentes van en `agents/`
- Skills van en `skills/`

### 4. Commit y push

```bash
git add .
git commit -m "feat: descripción clara del cambio"
git push origin feature/mi-nueva-feature
```

### 5. Crear Pull Request

- Ve a GitHub y creá un PR desde tu rama hacia `main`
- Describí qué cambiaste y por qué
- Esperá la revisión

## Guía de estilo

- Archivos en markdown (`.md`)
- Nombres en inglés para archivos, contenido puede ser en español
- Un skill por carpeta con su `SKILL.md`
- Comandos y agentes como archivos `.md` individuales

## Qué tipo de contribuciones aceptamos

- Nuevos skills
- Mejoras a comandos existentes
- Corrección de errores
- Mejoras de documentación
- Nuevos agentes

## Proceso de revisión

Todas las contribuciones son revisadas por [@diegoavarela](https://github.com/diegoavarela) antes de ser mergeadas.

## Código de conducta

- Sé respetuoso
- Acepta feedback constructivo
- Enfocate en mejorar el proyecto

## Preguntas

Si tenés dudas, abrí un [Issue](https://github.com/diegoavarela/kaostc-plugin/issues).
